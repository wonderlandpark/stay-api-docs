Stay API

이 페이지는 StayCute Official API입니다. 유저의 제한된 권한에 따라 사용할 수 있는 API가 제한됩니다.

  • 글로벌 밴(Global Ban)
    • Base URL
    • /check/user
    • /add/user

글로벌 밴(Global Ban)

기본 URL

API의 기본 URL(Base URL)입니다.


/check/user

글로벌 밴 여부를 확인합니다.

GET
Headers
token 엑세스 토큰 or 일반 토큰
Parameter
userID 디스코드 유저 아이디 (19자리)

/add/user

유저를 글로벌 밴 합니다.

GET
Headers
token 엑세스 토큰 or 일반 토큰
Parameter
userID 디스코드 유저 아이디 (19자리)
Reason 글로벌 밴 사유 (옵션)
